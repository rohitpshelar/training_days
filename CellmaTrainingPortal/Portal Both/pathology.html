Pathology

The Pathology module allows laboratories to electronically process and report all tests requested from healthcare professionals within or outside of their network enabling the creation of a paperless or paper-light environment.


Pathology

The Pathology module allows laboratories to electronically process and report all tests requested from healthcare professionals within or outside of their network enabling the creation of a paperless or paper-light environment.

How to use Pathology module

You can learn "How to use Pathology module" here.

Workflow for Pathology
 

Login with username and password
Search and Select patient
Add Lab Investigation
Fill in the details, select any outstanding investigations
Create lab request
Print Lab Request
Take Sample, Add Label and Send to Lab
Log Sample into Lab
Start Analysis
Set lab result
Create Lab Results Letter
Print Label Sample and Selected External report
Logout
