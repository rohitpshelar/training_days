Workflow for Business intelligence & reporting
 

Login with username and password
Select Advance Reporting Search module from the home screen
By default you will be on Advanced Patient Search page
Select the filter options as per required and click on Execute button, which will then generate a pop-up whether to Execute a query to get the results or Save and Run Later
Save and Run Later option is provided to get the results later, for that specify the query name and description and Save it
Click on Edit button to change the selection critera anytime
Click on New Search button to execute a fresh query
To get the Statutory Reports, click on Statutory Reports section from the left pane
Select the year, Quarter and Report type from the drop down and click on Search button to get the results
To get the User Report, click on User Reporting section from left pane
Search with Username and date range to get the activity report of the user and also can use Patient Barcode field to get the user report for any specific Patient
To get the Pharmacy Report, click on Pharmacy Reporting section from left pane
Select the the date filter, medication status, prescription, medication name to get the results
To get the Financial report, click on Finance Reporting section from left pane
Select the Finance Report type from the drop down, select date range and click on search to get the results
