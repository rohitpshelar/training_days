Supplies & Inventory Management

Cellma's Supplies & Inventory Management module allows healthcare professionals to monitor all stock and inventory requirements across every clinical department in their organisation through customisable stock dashboards. All stock movements are documented in Cellma, preventing 'loss' of stock and automatically alerting users when low stock thresholds are reached.


Supplies & Inventory Management

Cellma's Supplies & Inventory Management module allows healthcare professionals to monitor all stock and inventory requirements across every clinical department in their organisation through customisable stock dashboards. All stock movements are documented in Cellma, preventing 'loss' of stock and automatically alerting users when low stock thresholds are reached.

How to use Supplies & Inventory Management module

1. You can learn "How to use Stock Transfer" here.

2. You can learn "How to use Stock Requisition" here.

Workflow for Supplies & Inventory Management
 

Login with username and password
Select Stock module from home screen
Select stock location as default pharmacy location
Search and Add Stock Item
Enter mandatory fields and click on save
Check Price by entering Quantity.Total Cost will display
Edit the Stock item by clicking on batch no. link after clicking on expand button.Change details and save
Search the same item and click on transfer link
Transfer Process:
Select the another pharmacy location as a request location.(Stock item will be transferred to requested location)
Go to transfers link and select the request location
Enter quantity to approve.Tick on added check box and click on
Tick on check box and create picklist
Enter picked quantity and click on save
Tick on check box and click on create dispatch order
Receive Process:Select Requested location by clicking on pharmacy link
Go to receiving link tick on the checkbox of transferred medication
Return Process:Search the same item and click on return link
Select the Default Pharmacy
Select the reason to Return
Enter amount to return
Click on Return button
Click on Return link and search medication
Click on Pending Return of same medication
Click on Return button
Logout
