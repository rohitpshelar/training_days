ePharmacy

Cellma's ePharmacy and Prescribing module ensures electronic transfer of prescriptions. This decreases the patients' waiting time and service print costs. A high level of satisfaction has been reported by organisations where Cellma is deployed. It has full integration capabilities and can be used as a standalone system or integrated seamlessly with other Cellma modules or third-party systems.

How to use ePharmacy module

You can learn "How to use ePharmacy module" here.

Workflow for ePharmacy
 

Login with username and password
Search and Select patient
Add Medication
Add medication to Prescription/Repeatable
Add Dose, Frequency, Method,Days, Add Variable Dose Regime, Max Refills etc.
Create Prescription
Dispense Medication
Print Labels
Collect the Medication
Create Invoice with dispensing cost
Logout
